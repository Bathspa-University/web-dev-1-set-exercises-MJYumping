HTML & CSS Quick Reference Guide

This webpage serves as a quick reference guide that showcases various tags and includes descriptions and code examples.

HTML TAGS 

<head> HEAD </head>

A head tag is a semantic tag containing metadata wherein character set, document title, and other metadata are defined. Also where a CSS file is connected into the main html document, and where custom fonts are imported to be called later on. 

                    
                        <head>
                            <meta charset="UTF-8">
                            <meta name="viewport" content="width=device-width, initial-scale=1.0">
                            <title>Document</title>
                        
                            <link rel="stylesheet" href="QUICKREFStylesheet.css">
                        
                            <link href="https://fonts.googleapis.com/css2?family=Rajdhani&display=swap" rel="stylesheet">
                        
                        </head>
                    
                

<body> BODY </body>

The body tag is where all parts of the html webpage is encoded. This includes the header, nav, body, footer, and other semantic tags. There may only be one body tag in an html document. 

                      
                        <body>

                            <div class="HeaderDiv">
                                <header>
                                    <h1 class="HeaderTitle"> HTML & CSS Quick Reference Guide </h1>
                        
                                    <div>
                                        <p class="webpagedesc"> "Example phrase here." </p>
                                            <hr class="hr">
                                    </div>
                                </header> 
                            </div>

                        </body>
                    
                

<header> HEADER </header>

A semantic tag, the header sets an introductory section where logos and navigational buttons and links are usually placed and helps a web programmer in organizing code. 

                      
                        <header>
                            <h1> Syzygy Inc. </h1>
                            <img src="vanillalogo.jpg" alt="logo">
                        </header>
                    
                

<nav> NAVIGATION </nav>

A nav tag is a semantic tag wherein you set navigation buttons/links that will direct to a separate page or a section further into the page. 

                      
                        <nav>
                            <a href="/www.facebook.com/"> Facebook </a>
                            <a href="/www.linkedin.com/"> LinkedIn </a>
                            <a href="/www.canva.com/"> Canva </a>
                        </nav>
                    
                

<main> MAIN </main>

A semantic tags that indicates the main content of the page. Can only be one main tag in an html document. Cannot be a descendant of or be within an <article>, <aside>, <footer>, <header>, or <nav> element. 

                      
                        <main>
                            <div class="sectionheader-semantic">
                                <h2> SEMANTIC TAGS </h2>
                            </div>
                        </main>
                    
                

<footer> FOOTER </footer>

Another semantic tag wherein together with the header, main, body, and 

                      
                        <footer>
                            <p> Coypright of MJ Corporation © </p>
                        </footer>
                    
                

<div> DIVISION </div>

A division or a div tag indicates that a new division of content is made. It a versatile tag wherein any kind of content can be inserted, like further divs which can be styled with CSS. 

                      
                        <div>
                            <div>
                                <h1> This is a header </h1>
                            </div>

                            <div>
                                <p> This is a paragraph </p>
                            </div>
                        </div>
                    
                

<table> TABLE </table>

HTML tag used to organize information into a table format. An html table is composed of table row <tr>, table header <th>, and table data <td> </td> 

                      
                        <table>
                            <tr>
                                <th> Student ID </th>
                            </tr>
                            
                            <tr>>
                                <td> 1234 </td>
                            </tr>
                        </table>
                    
                

<p> PARAGRAPH </p>

The paragraph tag indicates a paragraph element, and it is worth noting that browsers add a single blank line before and after each paragraph element. 

                      
                        <p> This is a paragraph </p>

CSS PROPERTIES 

CLASS AND ID

Classes and IDs work are methods of pinpointing which element in html a coder would like to modify in a CSS file. While classes can be applied to multiple elements, an ID is unique to an element and cannot be repeated. 

                      
                        HTML: 

                            <h1 class="Head"   id="Unique1" > This is a header with Class "Head" and ID "Unique1" </h1> 
                    
                

BOX MODEL CONCEPT

The box model concerns the concept of a hidden box that surrounds every html element. The measurements of said box can be modified in CSS. 

                      
                        
                    
                

PADDING

The padding property controls the spacing between the actual content of a box, and the border of the box. Padding can be coded in different formats.

  • Padding coded with 4 values indicate one by one adjusting of all 4 sides (top, right, bottom, left)
  • Coded with 3 values indicate adjusting of top, left and right together, and then bottom side
  • Coded with 2 values indicate adjusting of top and bottom together, then left and right together

                      
                        .box padding{

                            padding: 5px 5px 0px 58px;

                            padding: 5px 5px 0px;

                            padding: 5px 5px;
                        }
                    
                

BORDER

The border property sets the characteristics of the box border, from width, color, corner radius, to which border is visible.

  • Border is shorthand and depending on what type of characteristic is inputted will modify width or color.
  • Border-style sets the style of the border line, ranging from dotted, dashed. solid, etc.

                      
                        .box border{

                            border: 5px 5px 5px 0px;
                            
                            border: aquamarine;

                            border-style: dashed
                        }     
                    
                

MARGIN

The margin property sets the distance from a box border to an invisible line where the edge of other boxes will start.

  • Margin values can be set with specific pixel numbers, a percentage of the containing element, auto, wherein margin is automatically calculated, or inherit which makes the element inherit the margin of the containing element.
  • Margin set with 4 values indicate one by one adjusting of all 4 sides (top, right, bottom, left)
  • Coded with 3 values indicate adjusting of top, left and right together, and then bottom side
  • Coded with 2 values indicate adjusting of top and bottom together, then left and right together

                      
                        box margin{

                            margin: 4px 4px 4px 4px;

                            margin: auto;

                            margin: inherit;

                            margin 10% 50% 20% 40%;
                        }
                    
                

BACKGROUND-COLOR

The background-color property changes the color of an element.

    Color can be specified using:

  • Color names which are commonly supported by different browsers (aliceblue, darkolivegreen, darkviolet, etc.)
  • Hex value (#8B0000)
  • RGB rgb(80, 110, 70)
  • HSL hsl(89, 43%, 51%)
  • With rgb and hsl having an optional "alpha" value

                      
                       box backgroundcolor{

                        background-color: darkolivegreen;

                        background-color: #FFFFFF;

                        background-color: rgb(100, 255, 100);

                        background-color: hsl(89, 43%, 51%);
                       }
                    
                

FONT

Font property modifies the style, weight, size/line-height, and family of the font. 

                      
                        box font{

                            font: italic small-caps bold 12px/30px Georgia, serif;
                        }
                    
                

ALIGN

The align property sets the alignment of items within a div element. 

                      
                        .box alignment{

                            display: flex;

                            align-items: center;
                        }
                    
                

COLOR

The color property changes the color of text within an element. 

                      
                        .box text color{

                            color: #FFFFFF;
                        }
                    
                

BOX-SHADOW

The box-shadow property adds and modifies a shadow cast by the element to indicate 3D effects styling. 

                      
                        .box box-shadow{
                            
                            box-shadow: 0px 1px 4px 0px #888888;
                        }
